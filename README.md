# LinkFree

LinkFree is a platform where users can create and share their personalized link trees, providing a hub for all their important links. Built with the MERN stack, it offers a seamless and customizable experience.

## Features

- **User Authentication**: Secure user registration and login with JWT-based authentication.

- **Customizable Links**: Users can add, edit, and remove links, creating a personalized link tree.

- **Responsive Design**: A user-friendly and responsive design to ensure a seamless experience on various devices.

## Technologies Used

- **MongoDB**: NoSQL database for storing user data and link information.

- **Express.js**: Backend framework for handling API requests.

- **React.js**: Frontend library for building a dynamic user interface.

- **Node.js**: Server-side runtime for JavaScript.

- **Tailwind CSS**: Utility-first CSS framework for styling.

## Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/enn-dee/LinkFree
   cd LinkFree
   ```
2. Install dependencies:
```bash
npm install
```
3. Configure environment variables:

Create a .env file in the root directory with the following:
```
.env
MONGODB_URL=your_mongodb_connection_url
JWT_SECRET=your_jwt_secret
```
4. Run the application:

```bash
npm run server
```
